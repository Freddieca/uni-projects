Guide to Sync Files Between Laptop and Desktop Using Git

1. Install Git

Make sure Git is installed on both your laptop and desktop. Download Git.

2. Create a Remote Repository

  • On GitHub, GitLab, or another Git hosting service, create a new repository (public or private).
  • Copy the repository URL (e.g., https://github.com/username/repository.git).

3. Initialize the Repository Locally (on your laptop)

  1. Open your terminal.
  2. Navigate to the project folder and initialize Git: 
    cd /path/to/your/project
git init
  3. Link it to the remote repository: 
    git remote add origin https://github.com/username/repository.git

4. Add, Commit, and Push Changes

  • Stage and commit your changes: 
    git add .
git commit -m "Initial commit"
  • Push to the remote repository: 
    git push -u origin main

5. Clone on Your Desktop

  • On your desktop, open the terminal and navigate to where you want the project to be.
  • Clone the repository: 
    git clone https://github.com/username/repository.git

6. Syncing Changes Between Laptop and Desktop

Push Changes from Laptop

When you make updates, add, commit, and push:

git add .
git commit -m "Describe your changes"
git push

Pull Changes on Desktop

When you switch to your desktop, fetch the latest changes:

git pull

7. Repeat for Each Device

On each device:

  • Use git add, commit, and push to update the repository.
  • On the other device, use git pull to bring in the latest updates.

Summary Commands

# On laptop/desktop when making changes
git add .
git commit -m "Your message"
git push

# On the other device
git pull

This setup ensures smooth syncing of project files across devices.
